# SensorTech-Miniproject
This project presents a compact, handheld metal detection system built using an Arduino-based circuit. The device is designed to identify the presence of nearby metallic objects and provide immediate feedback through LEDs and a buzzer. It uses a simple, low-power design powered by a portable power bank via the Arduino board, making it practical for demonstrations and real-world portability. The system also includes basic differentiation between types of metals based on how they influence the sensor’s behavior.
The working principle is based on electromagnetic induction. A coil generates a magnetic field when current flows through it, and any nearby metal object alters this field by changing the coil’s inductance. The circuit measures these small changes indirectly by sending pulses through the coil, capturing the resulting signal on a capacitor, and reading it using the Arduino’s analog input. By repeatedly sampling and averaging the readings, the system detects even minor variations and compares them to a baseline to determine when metal is present.
